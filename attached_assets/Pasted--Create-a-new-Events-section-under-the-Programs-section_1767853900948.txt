“Create a new ‘Events’ section under the ‘Programs’ section on the main website. This section should highlight upcoming and recent Risin-led ecosystem events in a clean, modern, and highly compact format.

Section Purpose:

Showcase curated, high-quality events run or supported by Risin Ventures.

Encourage immediate action (RSVP / Learn More).

Visually communicate momentum and ecosystem activity without overwhelming the page.

Layout & Structure:

Use a horizontal event strip design.

Display one event at a time within the strip.

If there are two or more events, enable a horizontal slider / carousel.

The section must be narrow in height while still containing all essential information.

Avoid tall cards or stacked layouts.

Single Event Content (One View):
Each event strip should include:

Event title

Short description (maximum 1–2 lines)

Date and time

Location

Small visual element (subtle banner or thumbnail — not dominant)

One primary CTA button (e.g., ‘RSVP Now’, ‘Register’, or ‘Learn More’)

Interaction & Navigation:

Desktop: arrow navigation or subtle pagination dots.

Mobile: smooth left-to-right swipe interaction.

No autoplay, no aggressive animations.

Transitions must feel frictionless and non-distracting.

Responsive Behavior:

Mobile view should maintain the single-event horizontal layout.

Events should scroll horizontally, similar to the Portfolio section behavior.

Text must truncate gracefully without increasing vertical height.

Design Guidelines:

Match the overall Risin Ventures visual language.

Keep typography compact, confident, and readable.

Use whitespace intentionally but avoid large empty gaps.

CTA must be clearly visible at all times.

Maintain consistency with Programs and Portfolio components.

Content Example – Generate One Event Using the Following Details:

Event Name: SustAInova Roadshow – Doha

Theme: Intersection of AI & Sustainability

Date: 21st January 2026

Time: 4:00 PM – 6:00 PM

Location: QSTP, Doha

Format: Invite-only curated roundtable

CTA Button: RSVP Now

Design Intent:

The Events section should feel like a live pulse of the ecosystem, not a marketing banner.

Emphasize credibility, curation, and action.
